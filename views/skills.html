On top of the basics, such as HTML5, CSS, and JavaScript, I have the most experience with the MEAN stack: MongoDB, Express.js, Angular.js, and Node.js. When working with CSS, I usually use Compass (SASS based) because it allows nesting, variables, mixins and selector inheritence. It is easier to organize and maintin stylesheets. It saves me time (and headache). I also like using Bootstrap to take advantage of its default CSS settings and compoments. HTML elements, such as input, form, button, and table, are just prettier with Bootstrap. Although Bootstrap's reponsive utilities are useful, once I learned about Flexbox, CSS3's new layout mode, it has replaced Bootstrap's 'col-*-'s as my go-to method of creating a fully responsive application. I love Angular. When I first learned to use Angular, it was such a drastic change, it almost felt like when you go from the Stone Age to the Iron Age in Age of Empires 2. I have built a few single page applications with Angular (including this portfolio). One thing with Angular that I did not quite like was that it is not an ideal match with Bootstrap, (because its 'live' actions are based on jQuery plugins), but there is a decent alternative, UI Bootstrap. UI Router is awesome too, for allowing me to have more than one view and even nest them as I wish. Python, who doesn't love Python? I like how simple the syntax is. It looks clean, makes code aesthetically more pleasing to my eyes. I don't have to worry about balancing braces, as long as I keep indentation properly.
